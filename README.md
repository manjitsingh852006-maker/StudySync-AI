# 📚 StudySync AI

StudySync AI is an AI-powered learning assistant built to help students study smarter, not harder. It can generate quizzes, summarize content, and assist in learning through AI.

---

## 🚀 Features

- 🧠 AI-powered Quiz Generator  
- 📝 Smart Notes & Summary Generator  
- 📊 Clean and simple student-friendly UI  
- ⚡ Fast backend API integration  
- 🔍 Easy input-based learning system  

---

## 🛠️ Tech Stack

**Frontend:**
- React.js
- HTML, CSS, JavaScript

**Backend:**
- Node.js
- Express.js

**AI Integration:**
- OpenAI API / LLM-based model

## ⚙️ How to Run Locally

### 1. Clone the repository
```bash
git clone https://github.com/manjitsingh852006-maker/StudySync-AI.git
```

### 2. Install frontend dependencies
```bash
cd client
npm install
npm start
```

### 3. Install backend dependencies
```bash
cd server
npm install
node index.js
```
